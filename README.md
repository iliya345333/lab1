# Лабораторная работа №1

## Основные задания
- Разработка консольной стратегии в духе Heroes of Might and Magic.
- Реализация боевой карты с героями, юнитами и замками.
- Использование принципов ООП: инкапсуляция, наследование, полиморфизм.
- Работа с игровыми сущностями: герои, юниты, строения.
- Поддержка сражений и перемещений на карте.

## Индивидуальное задание
Сделать так, чтобы паладин мог лечить союзников.
